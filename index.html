Spesifikasi Final Project

Simulasikan pekerjaan data analyst sungguhan dari awal sampai akhir: satu studi kasus bisnis, dikerjakan end-to-end dengan semua tools yang sudah kamu pelajari.

STEP 1

Tentukan pertanyaan bisnis

Pilih dataset publik dari Kaggle (misal: data penjualan e-commerce, data HR, atau data marketing). Tentukan 3-5 pertanyaan bisnis nyata, contoh: "Produk apa yang paling menguntungkan?" atau "Kenapa karyawan banyak resign di divisi tertentu?"

STEP 2

Ambil & bersihkan data

Gunakan SQL untuk mengambil/menyaring data, lalu Python (Pandas) untuk membersihkan data kotor: missing values, duplikat, format tidak konsisten.

STEP 3

Analisis

Lakukan exploratory data analysis dengan Python/Excel: cari pola, tren, dan anomali yang relevan dengan pertanyaan bisnis di Step 1.

STEP 4

Visualisasi

Buat dashboard interaktif di Tableau Public atau Power BI yang menjawab pertanyaan bisnis dengan jelas, lengkap dengan KPI utama.

STEP 5

Insight & rekomendasi

Tulis laporan singkat (1-2 halaman): temuan utama + rekomendasi bisnis yang actionable, seolah-olah kamu presentasi ke manajer.

STEP 6

Publikasikan

Upload seluruh project ke GitHub (kode + dashboard + laporan) dengan README yang menjelaskan proses berpikirmu dari awal sampai akhir.

Deliverables yang harus ada:
  • File SQL berisi query yang dipakai untuk mengambil/menyaring data
  • Notebook Python (.ipynb) berisi proses cleaning & analisis data
  • Dashboard interaktif (link Tableau Public atau file Power BI)
  • Laporan insight & rekomendasi (PDF atau slide singkat)
  • README di GitHub yang menceritakan proses dan hasil project

tips

Cara belajar biar tidak nyerah di tengah jalan

  • Sisihkan waktu belajar yang sama tiap hari, minimal 1-2 jam, lebih baik dari belajar maraton lalu vakum berminggu-minggu
  • Jangan tunggu "paham 100%" baru praktik — praktik dulu sambil error, baru cari tahu kenapa
  • Gabung komunitas data (Kaggle, grup Telegram/Discord data analyst Indonesia) untuk tanya-jawab dan motivasi

Modal awal (gratis semua)

  • Laptop dengan RAM minimal 4-8GB sudah cukup untuk semua tools di roadmap ini
  • Koneksi internet untuk akses tutorial, dataset, dan tools cloud (Google Sheets, Tableau Public)
  • Akun Kaggle, GitHub, dan LinkedIn — semuanya gratis dan wajib punya sebagai data analyst
